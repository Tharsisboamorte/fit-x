# FIT-X
## Objetivo:
### Aplicativo com o objetivo de documentar os dias de exercicio fisico do cliente para finalidade de gerenciamento
## Tecnologias utilizadas:
- Firebase
- Firestore
- storage
- jetpack compose
- Android Studio
